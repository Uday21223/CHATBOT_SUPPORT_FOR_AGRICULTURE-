# Chatbot Support For Agriculture 
<img src="https://www.lifeofautomation.com/wp-content/uploads/2020/05/adobestock_194734459-1-2048x1365.jpeg" id="img" width="100%" height="auto">



The **Agriculture Chatbot** provides instant support and information for farmers and agricultural enthusiasts. It helps users by answering questions and offering advice on various agricultural topics.

## Key Features
- **Crop Information**: Learn about planting, growth conditions, and pest management.
- **Weather Updates**: Get real-time weather forecasts to plan farming activities.
- **Soil Health Tips**: Discover practices for maintaining soil fertility.
- **Irrigation Advice**: Find best practices for effective irrigation.
- **Market Prices**: Stay updated on current crop prices.
- **Sustainability Practices**: Explore eco-friendly farming methods.

## Tools Used
- **Flask**: For the backend.
- **Natural Language Processing (NLP)**: To understand user queries.
- **Machine Learning**: To improve chatbot responses over time.
- **JavaScript**: For the user interface.

## How to Run
1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/agriculture-chatbot.git
    cd agriculture-chatbot
    ```
2. Create and activate a virtual environment:
    ```bash
    python3 -m venv venv
    . venv/bin/activate
    ```
3. Install the required dependencies:
    ```bash
    pip install Flask torch torchvision nltk
    ```
4. Download NLTK resources:
    ```python
    import nltk
    nltk.download('punkt')
    ```
5. Modify `intents.json` for your chatbot's intents and responses.
6. Train the model:
    ```bash
    python train.py
    ```
7. Run the chatbot:
    ```bash
    python app.py
    ```
